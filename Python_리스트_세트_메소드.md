# Python 리스트/세트 주요 메소드

---

## 리스트 관련 주요 메소드

| 메소드 | 설명 |
|--------|------|
| **pop(위치)** | 리스트의 '위치'에 있는 값을 출력하고 해당 요소 삭제 |
| **extend(리스트)** | 리스트의 끝에 새로운 '리스트'를 추가하여 확장 |
| **reverse()** | 리스트의 순서를 역으로 뒤집음 |
| **sort()** | 리스트를 정렬, 기본값은 오름차순 |
| **index(값)** | 리스트에서 '값'이 저장된 요소의 위치 반환 |
| **count(값)** | 리스트에서 '값'이 저장되어 있는 요소들의 개수 반환 |
| **copy()** | 리스트를 복사 |

---

## 세트 관련 주요 메소드

| 메소드 | 설명 |
|--------|------|
| **pop()** | 세트의 값을 출력하고 요소를 삭제 (인수 X) |
| **add(값)** | 세트에 '값'을 추가함 |
| **update(세트)** | 세트에 새로운 '세트'를 추가하여 확장 |
| **remove(값)** | 세트에서 '값'을 찾아 해당 요소 삭제 |
