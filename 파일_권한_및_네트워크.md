# 파일의 권한 (Permission)

---

## 권한 종류

| 기호 | 의미 |
|------|------|
| **r** | Read (읽기) |
| **w** | Write (쓰기) |
| **x** | eXecute (실행) |
| **-** | 권한 없음 |

---

## 권한 구조

```
d | r w x | r - x | r - -     총 10자리
```

| 위치 | 의미 |
|------|------|
| 첫 번째 | **d** (디렉토리) 또는 **-** (파일) |
| 2~4번째 | **소유자 (Owner)** 권한 |
| 5~7번째 | **그룹 (Group)** 권한 |
| 8~10번째 | **기타사용자 (Other)** 권한 |

### 예시

- 사용자에게 읽기, 쓰기, 실행 권한 부여
- 그룹에게 읽기, 실행 권한 부여
- 기타 사용자에게 읽기 권한 부여

# NAT (Network Address Translation)

- 네트워크 주소 변환
- **1개의 정식 IP 주소**에서 다량의 **가상 사설 IP 주소를 할당 및 연결**하는 방식

# ATM (비동기 전송모드)

- 교환전화 등이 쓰이는 **회선교환**과 **패킷교환**의 장점을 결합한 교환 및 다중화 기술
- 모든 데이터를 셀로 분할하여 비동기식 시분할 다중화 방식으로 전송
- 셀은 **53byte**의 **고정길이**, 속도가 빠르고 효율이 좋다
- **CBR, VBR**의 처리가 가능하며 **B-ISDN**과 결합하여 서비스를 제공하기도 함
