# 파일의 권한 (Permission)

---

## 권한 종류

| 기호 | 의미 |
|------|------|
| **r** | Read (읽기) |
| **w** | Write (쓰기) |
| **x** | eXecute (실행) |
| **-** | 권한 없음 |

---

## 권한 구조

```
d | r w x | r - x | r - -     총 10자리
```

| 위치 | 의미 |
|------|------|
| 첫 번째 | **d** (디렉토리) 또는 **-** (파일) |
| 2~4번째 | **소유자 (Owner)** 권한 |
| 5~7번째 | **그룹 (Group)** 권한 |
| 8~10번째 | **기타사용자 (Other)** 권한 |

### 예시

- 사용자에게 읽기, 쓰기, 실행 권한 부여
- 그룹에게 읽기, 실행 권한 부여
- 기타 사용자에게 읽기 권한 부여
